== NetApp ONTAP

NetApp ONTAP 軟體是運作在 AFF、ASA 和 FAS 儲存系統上的作業系統。ONTAP 提供高度可擴充的儲存架構，可實現不中斷營運、不中斷升級和敏捷的資料基礎架構。

如需ONTAP 更多關於效能的資訊、請參閱 http://www.netapp.com/us/products/platform-os/ontap/index.aspx["產品頁面ONTAP"^]。
