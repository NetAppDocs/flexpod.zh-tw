= 部分技術規格FlexPod
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./../media/


[role="lead"]
FlexPod Express 是一種預先設計、符合最佳實務的架構，它是基於 Cisco Unified Computing System（Cisco UCS）和 Cisco Nexus 系列交換器所建置，儲存層則採用 NetApp AFF、ASA 或 FAS 入門級儲存系統。FlexPod Express 是一個適用於執行各種虛擬化 Hypervisor、裸機作業系統（OS）和企業工作負載的理想平台。

FlexPod Express 不僅提供基本配置，而且具有靈活性，可調整大小和最佳化，以適應許多不同的使用情境和要求。
