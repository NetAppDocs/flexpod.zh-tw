[role="lead"]
FlexPod Express 是一個預先設計、符合最佳實務的架構，它是基於 Cisco 統一運算系統 （Cisco UCS） 和 Cisco Nexus 系列交換器構建，儲存層則採用 NetApp FAS 或 NetApp E-Series 儲存。FlexPod Express 是一個適用於執行各種虛擬化 Hypervisor、裸機作業系統 （OS） 和企業級工作負載的理想平台。

FlexPod Express 不僅提供基本配置，還具有高度的靈活性，可根據不同的使用情境和需求進行調整和最佳化。本文檔依所使用的儲存系統對 FlexPod Express 配置進行分類，分為 FlexPod Express with NetApp FAS 和 FlexPod Express with E-Series 兩類。
