= 部分技術規格FlexPod
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./../media/


[role="lead"]
FlexPod Express 是一個預先設計、符合最佳實務的架構，它是基於 Cisco 統一運算系統 （Cisco UCS） 和 Cisco Nexus 系列交換器構建，儲存層則採用 NetApp AFF、ASA 或 FAS 入門級儲存系統。FlexPod Express 是一個適合在中小型環境、遠端辦公室和分公司 （ROBO） 部署以及邊緣位置執行各種虛擬化 Hypervisor、裸機作業系統 （OS） 和企業級工作負載的平台。

FlexPod Express 不僅提供基本配置，還具有高度的靈活性，可根據不同的使用情境和需求進行調整和最佳化。
