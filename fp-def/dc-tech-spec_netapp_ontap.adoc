= NetApp ONTAP
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./../media/


[role="lead"]
所有 NetApp FAS、AFF 和 ASA 系統都安裝了 NetApp ONTAP 軟體。FlexPod 已通過 ONTAP 軟體驗證，提供高度可擴充的儲存架構，可實現不中斷營運、不中斷升級和敏捷的資料基礎架構。

如需ONTAP 更多關於效能的資訊、請參閱 http://www.netapp.com/us/products/data-management-software/ontap.aspx["資料管理軟體ONTAP"^] 產品頁面。
