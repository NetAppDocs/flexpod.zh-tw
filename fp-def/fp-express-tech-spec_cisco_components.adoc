== Cisco Nexus交換器

所有 FlexPod Express 架構都需要備援交換器。

FlexPod Express 搭配 NetApp AFF 或 FAS 架構，使用 Cisco Nexus 31108 交換器建置。FlexPod Express 搭配 Cisco UCS Mini （Cisco UCS 管理型）架構，已使用 Cisco Nexus 3524 交換器完成驗證。此組態也可使用標準交換器部署。

FlexPod Express with E-Series 可以與標準交換器一起部署。

下表列出 Cisco Nexus 系列機箱的零件號碼；其中不包括額外的 SFP 或附加模組。
