[[cisco-ucs-x-series-direct]]
== Cisco UCS X-Series Direct

Cisco UCS X-Series Direct 是一種新的架構，它將一對 Cisco UCS Fabric Interconnect 9108 100G 與 Cisco UCS X9508 機箱和 Cisco UCS X-Series 運算節點整合在一起，從而提供 UCS X-Series Direct 解決方案。

有關 Cisco UCS X 系列功能和特性的快速概述，請參閱link:https://www.cisco.com/c/en/us/products/collateral/servers-unified-computing/ucs-x-series-modular-system/ucs-x-series-direct-aag.html["Cisco UCS X-Series Direct 一覽"^]。

Cisco UCS X 系列 Direct 由下列元件組成：

* Cisco UCS 9108 100G Fabric Interconnect
* Cisco UCS X9508 機箱
* Cisco UCS X 系列運算節點
* 可選組件：
+
** PCIe 節點和 X-Fabric 模組
** 第二台 Cisco UCS X9508 機箱，配備 X-Series 運算節點、Cisco UCS 9108-100G 智慧交換矩陣模組以及最多四台 C-Series 機架式伺服器





NOTE: 請參閱此連結以取得 UCS X-Series Direct 的完整技術規格： https://www.cisco.com/c/en/us/products/collateral/servers-unified-computing/ucs-x-series-modular-system/ucs-x-series-direct-ds.html["Cisco UCS X 系列 Direct 資料表"]。

[[cisco-unified-edge]]
== Cisco Unified Edge

Cisco Unified Edge 是一款支援 AI 的系統，它融合了運算、網路、儲存和安全功能，重新定義了邊緣運算。其模組化設計面向未來、節能高效、易於維護，可針對當前工作負載和使用案例進行客製化，同時能夠適應快速發展的 AI 環境。

如需快速了解 Cisco Unified Edge 的功能與特性，請參閱link:https://www.cisco.com/c/en/us/products/collateral/servers-unified-computing/unified-edge/unified-edge-at-a-glance.html["Cisco Unified Edge 概覽。"^]

Cisco Unified Edge 由下列元件構成：

* Cisco UCS XE9305 機箱
* 兩台 Cisco Unified Edge Management Controllers，內建 25G 交換機
* 運算節點 - 選擇
+
** Cisco UCS XE130c M8 運算節點
** Cisco UCS XE150c M8 運算節點
