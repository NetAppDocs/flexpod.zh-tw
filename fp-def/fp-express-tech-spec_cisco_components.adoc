== Cisco UCS X-Series Direct

Cisco UCS X-Series Direct 是一種新的架構，它將一對 Cisco UCS Fabric Interconnect 9108 100G 與 Cisco UCS X9508 機箱和 Cisco UCS X-Series 運算節點整合在一起，從而提供 UCS X-Series Direct 解決方案。

Cisco UCS X 系列 Direct 由下列元件組成：

* Cisco UCS 9108 100G Fabric Interconnect
* Cisco UCS X9508 機箱
* Cisco UCS X 系列運算節點
* 選用元件：PCIe 節點和 X-Fabric 模組



NOTE: 有關 UCS X-Direct 的完整技術規格，請參閱此連結： https://www.cisco.com/c/en/us/products/collateral/servers-unified-computing/ucs-x-series-modular-system/ucs-x-series-direct-ds.html["Cisco UCS X 系列 Direct 資料表"]。
