== NetApp磁碟櫃與磁碟



== NetApp FAS

儲存控制器至少需要一個 NetApp 磁碟櫃。所選的 NetApp 磁碟櫃類型決定了該磁碟櫃內可用的磁碟機類型。

FAS2700 和 FAS2600 系列控制器提供包含雙儲存控制器和硬碟的整合配置，所有硬碟均安裝在同一機殼內。此配置支援 SATA 或 SAS 硬碟；因此，除非效能或容量需求需要更多硬碟位，否則無需額外的外部硬碟架。

[NOTE]
====
所有磁碟架零件編號均指配備兩個 AC PSU 的空磁碟架。如需其他零件編號，請諮詢您的銷售代表。

磁碟機的零件號碼會根據您購買的磁碟尺寸和外形規格而有所不同。請諮詢您的銷售代表以取得更多零件號碼。

====

=== E系列儲存
