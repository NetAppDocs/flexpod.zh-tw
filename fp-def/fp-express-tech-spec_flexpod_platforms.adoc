= 平台FlexPod
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./../media/


[role="lead"]
共有兩FlexPod 個支援平台：

* *FlexPod Datacenter。*該平台是一個高度可擴展的資料中心基礎架構，適用於企業工作負載和應用程式、虛擬化、虛擬桌面基礎架構 （VDI） 以及公有雲和私有雲。FlexPod Datacenter 有其自身的規範，這些規範已記錄在 https://docs.netapp.com/us-en/flexpod/fp-def/dc-tech-spec_solution_overview.html["資料中心技術規格FlexPod"^]。
* *《Express》*此平台是一款精巧的融合式基礎架構、專為遠端辦公室和邊緣使用案例而設計。FlexPod
