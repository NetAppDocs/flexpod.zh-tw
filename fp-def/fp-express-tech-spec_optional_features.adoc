== iSCSI開機選項

iSCSI 啟動選項的最低要求包括：

* 在NetApp儲存控制器上啟動的iSCSI授權/功能
* NetApp儲存控制器HA配對中每個節點上都有一個雙埠10Gbps乙太網路介面卡
* Cisco UCS伺服器中能夠開機的介面卡




== Cisco UCS X-Series Direct 和直連 AFF/ ASA/ FAS 選項

NetApp 儲存控制器可以直接連接到 Cisco UCS 光纖互連，無需任何上游 SAN 交換器。FlexPod Express 支援使用 UCS X-Series Direct 進行 FC 和 iSCSI 開機。

授權與硬體需求如下：

* 需要NetApp儲存控制器上的傳輸協定授權。
* 伺服器需要Cisco UCS介面卡（啟動器）。如需支援的Cisco UCS介面卡清單、請參閱NetApp http://mysupport.netapp.com/matrix["IMT"^]。
* 需要NetApp儲存控制器上的目標介面卡。


下圖顯示了基於 IP 的 SAN 的 UCS X-Series Direct 和直接連接的 ASA 拓撲。

image:fp-express-tech-spec_image10.png["UCS X-Series Direct 和直接連接 ASA 拓撲，適用於基於 IP 的 SAN"]

* 附註： *

* Cisco UCS配置為乙太網路交換模式。
* 從目標到光纖互連的iSCSI連接埠、會設定為iSCSI資料的乙太網路儲存連接埠。


下圖顯示了適用於 IP 型和 FC 型 SAN 的 UCS X-Series Direct 及直接連接 ASA 拓撲。

image:fp-express-tech-spec_image11.png["UCS X-Series Direct 和直接連接 ASA 拓撲，適用於以 IP 為基礎和以 FC 為基礎的 SAN"]

* 附註： *

* Cisco UCS配置為FC交換模式。
* 從目標到架構互連的FC連接埠會設定為FC儲存連接埠。


有關此組態的詳細資訊，請參閱 https://www.netapp.com/pdf.html?item=/media/135828-flexpod-san-ucs-xdirect-asa-nva-deploy-guide.pdf["FlexPod 採用 Cisco UCS X-Series Direct 和 NetApp ASA 的 SAN 解決方案"^]。


NOTE: 以上範例使用了 NetApp ASA 儲存控制器，但 NetApp AFF 和 FAS 儲存控制器也支援相同的設定。
