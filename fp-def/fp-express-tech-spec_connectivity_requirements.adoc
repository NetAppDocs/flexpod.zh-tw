== FlexPod Express 與 NetApp AFF/ASA/FAS 的連線要求

FlexPod Express 與 NetApp AFF/ ASA/ FAS 的連線需求通常包括：

* NetApp 儲存控制器必須直接連接到 Cisco Nexus 交換器。
* 對於 Cisco UCS X-Series Direct 部署，NetApp 儲存控制器可以直接連接到嵌入式 Fabric Interconnects。
* 核心FlexPod 不需在核心不整元件之間放置額外設備。
* 需要使用虛擬連接埠通道 (vPC) 將 Cisco Nexus 9000 系列交換器連接到 NetApp 儲存控制器。
* 雖然這不是必要的、但建議在整個環境中啟用巨型框架支援。
