== FlexPod Express with NetApp AFF 或 FAS 的軟體需求

FlexPod Express with NetApp FAS 的軟體需求包括：

* ONTAP 9.1 或更新版本
* Cisco NX-OS 版本 7.0(3)I6(1) 或更高版本
* 在 Cisco UCS 管理的設定中，Cisco UCS Manager UCS 4.0(1b)


所有軟體都必須在 http://mysupport.netapp.com/matrix["NetApp IMT"^]中列出並獲得支援。某些軟體功能可能需要比先前架構中列出的最低版本更新的程式碼版本。
