FlexPod 設計支援靈活的基礎架構，涵蓋各種硬體元件和軟體版本。

請參考規則集建置或組裝有效的 FlexPod 組態。本文件中列出的數字和規則代表 FlexPod 的最低需求，可根據不同環境和使用案例的需要，透過使用所包含產品系列中的支援產品進行擴充。
