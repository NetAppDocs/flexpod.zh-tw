= 結論
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./../media/


不同組織、不同業務單位、甚至組織內不同業務單位之間的雲端發展歷程各不相同。有些人選擇快速採用、有些則採取較保守的方法。無論組織規模大小、雲端採用速度如何、均可將其納入雲端策略、進一步展現出效益與擴充性優勢。FabricPool FlexPod
