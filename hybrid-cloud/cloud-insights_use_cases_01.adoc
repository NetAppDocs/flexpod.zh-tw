[[digital-advisor-integration]]
== Digital Advisor 整合

Cloud Insights 已完全整合至 Digital Advisor 儲存監控平台中。部署為 FlexPod Datacenter 解決方案一部分的 ONTAP 系統，會自動設定為透過每個系統內建的 AutoSupport 功能，將資訊傳回給 NetApp。這些報告會定期產生，或在系統偵測到故障時動態產生。透過 AutoSupport 傳輸的資料會進行彙整，並顯示在 Cloud Insights 中 Digital Advisor 功能表下易於存取的儀表板中。

若要透過 Cloud Insights 控制面板存取 Digital Advisor 資訊，請完成以下步驟：

. 按一下左側「管理」功能表下的「資料收集器」選項。
+
image:cloud-insights_image13.png[""]

+
image:cloud-insights_image23.png[""]

. 按一下資料收集器、即可取得該收集器所監控的環境和裝置摘要。

在靠近底部的裝置清單下、按一下ONTAP 所監控的功能表儲存系統名稱。這會顯示收集到的系統相關資訊儀表板、包括下列詳細資料：

. 若要開啟新的瀏覽器索引標籤並前往風險緩解頁面（該頁面會顯示哪些節點受到影響、風險的嚴重程度，以及需要採取哪些適當措施來修正已識別的問題），請按一下 Digital Advisor 的連結。

[[create-custom-dashboards]]
=== 建立自訂儀表板

除了使用顯示全系統資訊的預設儀表板之外、Cloud Insights 您也可以使用下列功能來建立完全自訂的儀表板：利用「效益FlexPod 分析中心」解決方案、您可以將焦點放在特定儲存Volume的資源使用上、 因此、部署在融合式基礎架構中的應用程式、必須仰賴這些磁碟區才能有效執行。這樣做有助於您更好地視覺化特定應用程式及其在資料中心環境中所耗用的資源。



==== 建立自訂儀表板以評估儲存資源

若要建立自訂儀表板以評估儲存資源、請完成下列步驟：

. 若要建立自訂儀表板、請將游標移至Cloud Insights 「資訊功能」主功能表上的「儀表板」、然後在下拉式清單中按一下「+ New Dashboard（+新儀表板）」。
+
image:cloud-insights_image31.png[""]

+
「新儀表板」視窗隨即開啟。

. 命名儀表板、然後選取用於顯示資料的小工具類型。您可以從多種圖表類型、甚至是備註或表格類型中選取、以顯示所收集的資料。
+
image:cloud-insights_image32.png[""]

. 從「新增變數」功能表中選擇自訂變數。
+
如此一來、所呈現的資料就能集中顯示更多特定或專業因素。

+
image:cloud-insights_image33.png[""]

. 若要建立自訂儀表板、請選取您想要使用的小工具類型、例如圓形圖、以依磁碟區顯示儲存使用率：
+
.. 從「新增小工具」下拉式清單中選取「圓形圖」小工具。
.. 使用描述性識別碼來命名小工具、例如「已使用容量」。
.. 選取您要顯示的物件。例如、您可以依主要詞彙Volume搜尋、然後選取「volume.performance.capacity.used`」。
.. 若要依儲存系統進行篩選、請使用篩選器、並在FlexPod 「Datacenter解決方案」中輸入儲存系統的名稱。
.. 自訂要顯示的資訊。依預設、此選項會顯示ONTAP 「不實資料量」、並列出前10名。
.. 若要儲存自訂儀表板、請按一下「儲存」。
+
image:cloud-insights_image34.png[""]

+
儲存自訂Widget之後、瀏覽器會返回「新儀表板」頁面、顯示新建立的Widget、並允許採取互動式動作、例如修改資料輪詢期間。

+
image:cloud-insights_image35.png[""]







[[advanced-troubleshooting]]
=== 進階疑難排解

Cloud Insights 支援對 FlexPod 資料中心融合基礎架構中的任何儲存環境應用進階故障排除方法。它利用上述各項功能的組件：Digital Advisor 整合、包含即時統計資訊的預設儀表板以及自訂儀表板，能夠及早發現並快速解決可能出現的問題。透過 Digital Advisor 中的風險列表，客戶可以找到可能導致問題的已報告組態錯誤，以及可以修復這些問題的已報告漏洞和已修補的程式碼版本。觀察 Cloud Insights 主頁上的即時儀表板有助於發現系統效能模式，這些模式可能是問題出現的早期跡象，並有助於快速解決問題。最後，建立自訂儀表板的功能使客戶能夠專注於基礎架構中最關鍵的資產並直接監控這些資產，從而確保實現其業務連續性目標。



[[storage-optimization]]
=== 儲存最佳化

除了故障排除之外，還可以利用 Cloud Insights 收集的資料來最佳化部署在 FlexPod 資料中心融合基礎架構解決方案中的 ONTAP 儲存系統。如果某個磁碟區出現高延遲（可能是因為多個高效能需求的虛擬機器共用同一個資料存放區），則該資訊會顯示在 Cloud Insights 儀表板上。透過這些資訊，儲存管理員可以選擇將一個或多個虛擬機器移轉至其他磁碟區、在集合體層級之間或 ONTAP 儲存系統中的節點之間移轉儲存磁碟區，從而實現效能最佳化的環境。透過 Digital Advisor 與 Cloud Insights 整合所獲取的資訊，可以凸顯導致效能低於預期的組態問題，並提供建議的修正措施；若加以實施，即可修復任何問題，並確保儲存系統達到最佳調校狀態。
